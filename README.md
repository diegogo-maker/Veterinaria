# Veterinaria
Nombre del proyecto: Veterinaria

Se trata de una aplicacion para registrar perfiles y agendar citas en una veterinaria pudiendo registrar mascotas y dueños para un mismo perfil así como poner información detallada de cada uno y cambiar los datos a conveniencia.

El objetivo es crear un sistema que permita crear perfiles de clientes para una veterinaria de forma asíncrona y también se pueda agendar citas para cada cliente de forma no presencial.

Aplicación de una veterinaria
Funcionalidades principales:
-Inicio de sesion.
-Registro de un nuevo usuario.
-Edicion del perfil(asignar dueño o paciente y cambiarlo acorde a lo requerido).
-Agendar citas
-Consultar los detalles de una cita
-Cancelar una cita  
-Edición de una cita
-Consultar clientes
